# CAPLOL - Landing Page

O repositório dessa Landing Page é um projeto destinado à criação de uma página web moderna e atraente para divulgação e apresentação de informações sobre campeonatos. O objetivo principal desta landing page é fornecer uma experiência visualmente agradável e intuitiva para os visitantes, apresentando detalhes dos campeonatos, inscrições, resultados, equipes participantes e muito mais.
